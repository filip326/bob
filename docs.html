Grundbefehle

SCHRITT
Bob geht einen Schritt. Er bewegt sich um ein Kästchen in die Richtung in die er schaut.
SCHRITT
Bob kann keine Mauern passieren. Steht Bob vor einer Mauer oder dem Rand der Karte, gibt es einen Fehler.
LINKS-WENDUNG
Bob dreht sich um 90° nach links (aus seiner Sicht). So ändert sich die Blickrichtung von Süden nach Osten, Osten nach Norden u.s.w.
LINKS-WENDUNG
keine
PLATZIEREN
Bob platziert eine von maximal 8 Markierungen. Es erhöht sich die Anzahl von Markierungen um 1.
PLATZIEREN
Bob kann keine Markierung platzieren, wenn es bereits 8 Markierungen auf einem Feld gibt. Versuch Bob auf einem vollen Feld 8 Markierungen zu platzieren, gibt es einen Fehler.
AUFHEBEN
Bob hebt eine von maximal 8 Markierungen von einem Feld auf. Es verringert sich die Anzahl der Markierungen um 1.
AUFHEBEN
Bob kann keine Markierung aufheben, wenn es keine Markierung gibt. Versucht Bob auf einem Feld ohne Markierungen eine Markierung aufzuheben, gibt es einen Fehler.
