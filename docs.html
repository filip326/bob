AUFHEBEN

Bedingungen

Bedingungen
IST|NICHT <Bedingung>
In Bob.js gibt es Bedingungen, die geprüft werdne können. Eine Bedingung besteht aus IST oder NICHT und der zu prüfenden Tatsache. Ist die Tatsache wahr, ist die Bedingung erfüllt, sonst nicht. Lautet die Bedingung NICHT, umgekehrt. Bedingungen können in SCHLEIFEN und WENN-ABFRAGEN genutzt werden.
IST WAND
NICHT HAUS
WAND
Bob schaut, ob vor ihm eine Wand ist. Ist vor ihm eine Wand, wahr, kann er einen Schritt machen, falsch. Der Rand / das Ende der Welt zählt auch als Wand.
HAUS
Bob schaut, ob er zu Hause ist. Ist er zu Hause wahr, sonst falsch.
NORDEN | SÜDEN | OSTEN | WESTEN
Bob schaut, ob er in eine der 4 Richtungen schaut. Schaut er in die richtige Richtung wahr, sonst falsch.
ZUFALL
Gibt in ungefähr 50% der Aufrufe wahr zurück, sonst falsch. Es wird pseudo-Zufall verwendet.

WENN <Bedingung>
<Befehle 1. Codeabschnitt>

Welcher Code ist der schönere? Wiederhole X-Mal schleifen verwendest du, wenn du im Vorhinein weißt, wie oft eine Sequenz ausgeführt werden soll. Der Codeabschnitt wird so oft ausgeführt, wie du es angibst.
WIEDERHOLE <Zahl>-MAL
<Befehle>
ENDE
Die Befehle werden X-Mal hintereinander wiederholt.
WIEDERHOLE SOLANGE
WIEDERHOLE SOLANGE <Bedingung>
<Befehle>
ENDE
Die Befehle werden so lange ausgeführt, solange die Bedingung erfüllt ist. Ist die Bedingung nicht erfüllt, wird der restliche Code weiter fortgesetzt / das Programm beendet.
Achte darauf, dass dein Programm nicht unendlich lange läuft. Deine Bedingung sollte nie unendlich lange laufen.

Methoden

LERNE
LERNE <Name>
<Befehle>
ENDE
Mit dem Lerne-Befehl lernt Bob etwas neues. Du kannst überall in deinem Code auf die gelernete Methode zugreifen, indem du sie wie einen anderen Befehl aufrufst.
<Name>
Der Name sollte nur Buchstaben, keine Zahlen oder Sonderzeichen enthalten.
Vermeide es, innerhalb einer Methode sich selbst aufzurufen. Dies kann zu einer unendlichen Schleife oder einer Überlastung des Speichers führen.

Debugging

ALERT
SAY <-- <Text>
Dieser Befehl gibt den gewünschten Text aus.
SAY <-- Hello World
LOG
LOG <-- <Text>
Dieser Befehl schreibt den gewünschten Text in die Konsole.
LOG <-- Hello World
TELEPORT
TP <-- <x> <-- <y>
Dieser Befehl platzier Bob sofort auf das gewümschte Feld. Die x und y Koordinaten müssen zwischen einschließlich 0 und 9 sein.
